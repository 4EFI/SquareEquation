\doxysubsection{Detailed Description}
Чтобы убрать логирование, нужно объявить макрос NLOG (\#define NLOG) 

Также в библиотеке есть макрос \$\$(code) (\#define \$\$(code)). Данный макрос логирует тот код и выполняет его, который передается ему в виде аргумента.

Макрос \$\+LOG\+\_\+\+LVL\+\_\+\+UP (\#define \$\+LOG\+\_\+\+LVL\+\_\+\+UP). Данный макрос повышает уровень логирования на 1, пока не выйдет из соответствующего блока, в этом случае уровень блока понижается на 1.
